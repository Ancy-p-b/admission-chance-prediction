# 🎓 Admission Chance Prediction System

## 📌 Overview

The **Admission Chance Prediction System** is a machine learning-based project that predicts the probability of a student's admission into a university based on various academic and profile factors.

This project helps students estimate their chances of getting admitted by analyzing parameters such as GRE score, TOEFL score, CGPA, SOP, LOR, and more.

---


## 🚀 Features

* Predicts admission chances using ML models
* User-friendly input system
* Data preprocessing and cleaning
* Uses TF-IDF / numerical transformation (if text involved)
* Accurate and efficient prediction system

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries Used:**

  * NumPy
  * Pandas
  * Scikit-learn
  * Matplotlib / Seaborn (optional for visualization)

---

## 📊 Input Parameters

The model uses the following inputs:

* GRE Score
* TOEFL Score
* University Rating
* SOP (Statement of Purpose)
* LOR (Letter of Recommendation)
* CGPA
* Research Experience

---

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/Ancy-p-b/admission-chance-prediction.git
```

2. Navigate to the project folder:

```bash
cd admission-chance-prediction
```

3. Install required libraries:

```bash
pip install -r requirements.txt
```

4. Run the project:

```bash
python main.py
```
